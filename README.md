# QR Code Generator 📱

</div>

An advanced QR Code Generator built with Python using the `qrcode` and `Pillow` libraries.

This project allows users to generate customized QR codes from text or URLs with different colors, custom file names, and automatic image preview support.

---

## 🚀 Features

- Generate QR codes from text or URLs
- Multiple QR color options
- Custom file naming
- Automatic image preview
- Error handling
- User-friendly terminal interface
- Supports multiple QR generations in one run

---

## 🛠️ Technologies Used

- Python
- qrcode
- Pillow (PIL)

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/nitheesh011109/python-qr-generator.git
```

Move into the project folder:

```bash
cd python-qr-generator
```

Install required libraries:

```bash
pip install qrcode[pil] pillow
```

---

## ▶️ How to Run

```bash
python main.py
```
